Sizlon operates a rule-based verification system.

System Structure

Input is accepted.
Specification is fixed.
Verification terminates.
Results are recorded.
The operating target is rule-conformance of documents and input.

What The System Does Not Do

The system does not make decisions.
The system does not provide recommendations.
The system does not interpret results.
The system does not optimize outcomes.

Responsibility Boundary
